# LoRaトランシーバの変更申請の届出方法
電子申請・届出システムLiteによる申請方法を以下に示します。

## 電波形式
電波の形式はF1Dとなります。

今回の申請では既に430MHz帯は包括記号「3VA」で免許を受けていたために変更なしでした。
単独で新規に申請されるケースは少ないと思いますがF1Dが入った型式(4VF等)を指定して下さい。
